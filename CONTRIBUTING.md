# Contributing to diffsan

The main contributor guide lives in [docs/contributing.md](docs/contributing.md) and is also published at <https://caika-lgtm.github.io/diffsan/contributing/>.

For local development, the short version is:

```bash
git clone https://github.com/caika-lgtm/diffsan.git
cd diffsan
make install
make verify
make test
```

Please read the full guide before opening a pull request.
